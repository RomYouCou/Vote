Setup

If you are seeing this message you have already setup your config file and installed dependencies. The next step is to customise your welcome page (this page), setup your account/admin privleges and add some questions for your users to vote on.

  1. User Account & Admin Privledges

    Click Login/Register in the top right then login/register using your prefered method.

    Visit /user.json, taking note of the id. Then in terminal run ./scripts/make-admin <user id>

  2. Add Polls

    Now you have admin permissions, you should be able to visit /admin/poll/new and be displayed with a new poll form.

  3. Edit Welcome Page

    Open /views/index.html in a text editor where you'll find the HTML for this page, update this to what you would like your visitors to see.
